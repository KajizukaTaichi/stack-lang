# Stack プログラミング言語
スタック指向型の強力なスクリプト言語

## 機能

1. スタック指向型の効率的な実行モデル
2. 明瞭で分かりやすいエレガントな構文
3. 便利で拡張性の高い豊富なコマンド
4. 強力かつ柔軟なデータ型の仕組み

## サンプルコード
これはStackプログラミング言語でFizzBuzz問題を解くサンプルコードです
```stack
(
    (i) var 
    (Fizz) () 
    i 3 mod 0 equal if
) (fizz) var

(
    (i) var 
    (Buzz) () 
    i 5 mod 0 equal if
) (buzz) var

1 101 1 range (i) (

    () (msg) var
    msg i fizz eval concat (msg) var
    msg i buzz eval concat (msg) var

    (
        i (msg) var
    ) ()
    msg () equal if
    msg print
) for
```
